# 海龟交易策略 — 突破入场 + ATR 动态止损 + 波动率仓位管理

> **核心结论**：海龟策略不是"更好的策略"，而是"更适合趋势市场的策略"。在趋势性强的标的上，其 ATR 止损机制使最大回撤显著低于双均线策略；在盘整市场中，反复的假突破和止损退出会持续侵蚀本金。

---

## 📖 Strategy Overview（策略概述）

- **策略类型**：趋势跟踪（Trend Following）
- **核心思想**：不预测市场方向，只跟随已经形成的趋势。价格突破 N 日高点时入场（趋势确认），跌破 M 日低点时离场（趋势反转）。仓位大小由波动率（N 值/ATR）标准化——波动大时自动减仓、波动小时自动加仓，每笔交易的亏损上限在入场前已被精确量化。支持金字塔加仓：趋势延续时价格每上涨 0.5×N 追加一个单位（最多 4 次），让利润在趋势中滚动。
- **历史背景**：Richard Dennis & William Eckhardt，1983 年。用 23 名零基础学员（"海龟"）在四年内创造超 1 亿美元利润，证明"交易可以被传授"。
- **适用市场**：趋势明确、波动率适中的市场。不适用于持续盘整/横盘市场。

## 📐 Mathematical Definition（数学定义）

### 唐奇安通道（Donchian Channel）
```
DC_High_t = max(High_{t-N+1}, ..., High_t)   // 上轨：N日最高价
DC_Low_t  = min(Low_{t-N+1}, ..., Low_t)     // 下轨：N日最低价
```
- 入场通道周期 N = 20（System 1）或 55（System 2）
- 离场通道周期 M = 10（System 1）或 20（System 2）

### N 值 / ATR
```
TR_t = max(High_t − Low_t, |High_t − Close_{t-1}|, |Low_t − Close_{t-1}|)
N_t  = WilderSmooth(TR, 20)_t = N_{t-1} × 19/20 + TR_t × 1/20
```
- Wilder 平滑 α = 1/N（非 EMA 的 2/(N+1)）

### 交易信号
```
入场: High_t > DC_entry.High_{t-1}  →  以 Close_t 买入
止损: Low_t ≤ StopPrice            →  以 StopPrice × (1−slippage) 卖出
离场: Low_t < DC_exit.Low_{t-1}    →  以 Close_t 卖出
```

### 仓位公式（严格推导）
```
RiskAmount = Cash × 2%                       // 这笔交易允许亏多少
StopDist   = 2.0 × N_t                       // 每股止损距离
Shares     = RiskAmount / StopDist            // 股数 = 允许亏损 ÷ 每股亏损距离
```
当止损价被触发时，实际亏损 ≈ RiskAmount ✓

### 金字塔加仓
```
加仓条件: Price_t ≥ LastAddPrice + pyramidAtr × N_t  (且 unitsHeld < maxUnits)
加仓数量: AddShares = RiskAmount / (atr_stop_mult × N_t)   // 与首仓公式一致
加仓后:   止损价收紧至 AddPrice − atr_stop_mult × N_t       // 保护已有利润
```
每笔加仓的风险预算与首仓相同，确保任何一次加仓触发止损时亏损可控。

### 动态止损
```
StopPrice = max(StopPrice, EntryPrice − 2×N_t)        // 基础止损
StopPrice = max(StopPrice, Close_t − 2×N_t)            // 移动止盈（只上移不下移）
```

## ⚙️ Parameters（参数说明）

| 参数 | 默认值 | 范围 | 作用 |
|------|--------|------|------|
| 入场通道周期 | 20 | 5–120 | 控制趋势确认的敏感度——越短信号越多、假突破越多 |
| 离场通道周期 | 10 | 3–60 | 控制趋势反转确认的速度——越短离场越快 |
| ATR 周期 | 20 | 5–60 | N 值计算窗口——影响仓位大小和止损距离 |
| 止损倍数 | 2.0 | 0.5–5.0 | 止损距离 = k×N——越小越容易被噪音震出 |
| 风险比例 | 2% | 0.5–5% | 单笔最大亏损占资金比例 |
| 初始资金 | 1,000,000 | — | 回测起始资金 |
| 手续费 | 0.03% | — | 买卖各收一次 |
| 滑点 | 0.01% | — | 买入上浮、卖出下浮 |
| 亏损过滤 | 开启 | — | 上一笔亏损则跳过下一次 System 1 信号 |
| 最大加仓次数 | 4 | 1–4 | 金字塔加仓单位数上限 |
| 加仓间隔 | 0.5×N | 0.3–1.0 | 价格需上涨 pyramidAtr×N 才触发加仓 |

## 📊 Backtest Results（回测结果）

**回测设置**：贵州茅台 (600519) / 2025-05-29 ~ 2026-07-03 / 初始资金 100 万 / 手续费 0.03% / 滑点 0.01%

| 指标 | 海龟 System 1 | 买入持有 | 双均线 MA5/MA15 |
|------|:-----------:|:------:|:-------------:|
| 累计回报 | −0.74% | −18.24% | −5.67% |
| 年化收益 | −0.47% | −11.63% | −3.61% |
| 最大回撤 | −9.72% | −21.35% | −15.20% |
| 夏普比率 | −0.29 | −1.12 | −0.58 |
| 胜率 | 40.0% | — | 35.7% |
| 交易次数 | 5 | — | 14 |

**解读**：在 13 个月的下跌市中，海龟和双均线都跑赢买入持有，但绝对收益均为负——说明趋势跟踪策略在单边下跌+仅做多的约束下无法盈利。海龟的 MDD（−9.72%）显著优于双均线（−15.20%）——ATR 止损机制确实起到了风险控制作用。

## 📈 Parameter Sensitivity（参数敏感性）

### 实验 A：通道周期

| 参数组 | 交易次数 | 累计回报 | 解读 |
|--------|:-----:|:------:|------|
| entry=10, exit=5 | 8 | −2.1% | 假突破多，手续费累积 |
| entry=20, exit=10 | 5 | −0.74% | 最佳平衡点 |
| entry=55, exit=20 | 2 | −1.5% | 信号太少，错过反弹 |

→ 通道周期过短（10 日）：假突破频繁，胜率下降，手续费侵蚀利润。通道周期过长（55 日）：入场过晚，捕获的趋势幅度不足。

### 实验 B：止损倍数

| 止损倍数 | MDD | 胜率 | 解读 |
|:------:|:---:|:--:|------|
| 1.5×N | −6.5% | 25% | 止损过紧，被噪音震出 |
| 2.0×N | −9.72% | 40% | 海龟默认，平衡 |
| 3.0×N | −14.2% | 60% | 回撤增大，几乎等于无止损 |

→ 2.0×N 是经验最优值——在"不被噪音震出"和"不在反转时损失过大"之间取得平衡。

## 💡 Key Findings（关键发现）

**什么时候有效**：
- 价格呈现明显的单向趋势（连续创新高或新低），ADX > 25
- 波动率适中——N/Price 在 1.5–3% 之间，提供足够的波动空间而不过于剧烈
- 趋势持续时间 ≥ 入场通道周期的 2–3 倍——如果趋势只持续 10 天但用 20 日通道入场，大概率在趋势结束时入场

**什么时候失效**：
- 持续盘整/横盘市场：价格反复突破通道上轨又回落 → 连续止损 → 本金被千刀万剐
- 跳空剧烈的标的：ATR 无法准确捕捉隔夜风险 → 止损价形同虚设 → 实际亏损远超预算
- 流动性差的标的：滑点占比上升 → 实际成交价远偏离信号价 → 回测结果与实盘差异大

**海龟 vs 双均线（Task3）**：
- 同为趋势跟踪策略，海龟的核心优势在于 **ATR 风险管理**——不是"入场更准"，而是"亏钱时亏得更少"
- 双均线信号更频繁（14 次 vs 5 次），但单笔风险管理缺失，MDD 更大
- 两策略在同一市场中高度相关——当趋势不存在时，两者都无法盈利

## 🚀 Future Improvements（改进方向）

1. **ADX 趋势过滤器**：ADX < 20 时暂停交易，仅在趋势确立时启用信号。可大幅减少盘整期的连续亏损。
2. **多时间框架确认**：System 1（20 日）信号触发时，检查 System 2（55 日）的趋势方向——两系统方向一致时加大仓位，不一致时减小仓位。
3. **动态参数**：根据市场波动率环境（ATR 百分位）自动调节通道周期——高波动期加长通道周期以减少假突破。
4. **允许做空**：当前仅做多。加入做空方向后，在下跌趋势中也能盈利，策略对称性显著提升。
5. **多资产组合**：同时应用于多只低相关性标的，通过分散化降低单标的连续亏损对总账户的影响——这也是海龟原版的核心理念（"多市场交易"）。
