AI量化交易课程项目总览

8 Tasks · GitHub Pages

关于本项目

本项目为北京大学光华管理学院商业分析工作坊《量化交易：AI大模型辅助的金融交易策略》的八个阶段性任务集合， 涵盖数据引擎搭建、技术指标构造、均线交叉策略、海龟交易法则、机器学习预测、智能交易策略、模拟实盘部署、以及最终策略路演。 数据源采用 AKShare 与 Tushare Pro 双通道，覆盖 A 股（沪深）与港股（联交所）共 10 只标的， 使用前复权（qfq）价格确保回测严谨性。看板均为自包含 HTML，无需后端即可在浏览器中交互使用。

Task 1
K线可视化
A股与港股10只标的的交互式K线图。支持市场切换、多股对比、十字光标、缩放拖拽，数据嵌入页面无需服务器。
进入看板
Task 2
技术指标分析
八大技术指标（RSI、MACD、布林带、ATR、KDJ、MA、CCI、ADX）的交互式计算与可视化，支持参数调节、多指标叠加对比。
进入看板
Task 3
双均线交叉策略
金叉买入、死叉卖出——最基础的趋势跟踪策略。交互式回测看板支持参数调节，全仓进出，对比买入持有基准。
进入看板
Task 4
海龟交易法则
复刻 Richard Dennis 的传奇策略：突破入场、ATR 动态止损、波动率仓位管理。回答"海龟在什么市场条件下有效"。
进入看板
Task 5
机器学习算法应用
线性回归、决策树、KNN等基础模型，scikit-learn训练与评估，RMSE、准确率、混淆矩阵分析。
即将发布
Task 6
ML驱动交易策略
扩展特征工程，训练模型预测涨跌概率，结合模型信号设计交易规则，与买入持有等朴素策略比较。
即将发布
Task 7
模拟实盘部署
JoinQuant平台基本操作，编写initialize和handle_data函数，运行模拟交易，记录订单日志与持仓变化。
即将发布
Task 8
报告撰写与策略路演
整合前序任务成果，撰写包含背景、方法论、代码片段、回测结果和结论反思的综合实习报告。
即将发布
