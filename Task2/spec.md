## 术语预定义

以下术语在指标公式中反复出现，在此统一定义：

| 术语 | 符号 | 定义 |
|------|------|------|
| **简单移动平均** | SMA(P, N) | 过去 N 期价格 P 的算术平均值：`SMA_t = (P_{t-N+1} + ... + P_t) / N`。每期权重相等 |
| **指数移动平均** | EMA(P, N) | 过去价格的加权平均，近期权重更大。递推公式：`EMA_t = P_t × α + EMA_{t-1} × (1-α)`，其中平滑系数 `α = 2/(N+1)`。N 称为"等效周期"——EMA 对价格变化的响应速度约等于同周期 SMA |
| **Wilder 平滑** | Wilder(P, N) | J.W. Wilder 使用的递推平滑公式：`W_t = P_t × (1/N) + W_{t-1} × (1 - 1/N)`。与 EMA 的区别在于平滑系数使用 `1/N` 而非 `2/(N+1)`，因此对相同 N 值，Wilder 平滑比 EMA 更缓慢。Wilder 的 N=14 在响应速度上约等于 EMA(N=27) |
| **标准差** | σ(P, N) | 过去 N 期价格 P 的标准差（Standard Deviation）：`σ = sqrt(sum((P_i - SMA_t)^2) / N)`，衡量价格围绕均值的离散程度 |
| **收盘价** | Close_t | 第 t 个交易日的收盘价，单位为元 |
| **最高价/最低价** | High_t / Low_t | 第 t 个交易日的最高成交价和最低成交价 |

## 1. RSI (相对强弱指数 / Relative Strength Index)

### 1.1 历史演化

**发明**: J. Welles Wilder Jr. 在 1978 年出版的《New Concepts in Technical Trading Systems》中提出 RSI。Wilder 是一名机械工程师出身，后转入房地产和期货交易。他的核心动机是构建一个**有界振荡器**——指标的取值范围固定在 0 到 100 之间，使得不同品种、不同时间段的读数可以直接比较。

**解决的核心问题**: 在 RSI 之前，常见的动量指标（如简单动量 = 今日收盘 − N 日前收盘）是无界的——无法回答"这个动量读数到底是高还是低"。RSI 通过将涨跌幅比例归一化到 [0, 100] 区间，首次提供了一个可横向比较的动量强度度量。

**参数 14 的来源**: Wilder 选择 N=14 并非基于统计优化，而是基于他观察的期货市场交易周期——14 个交易日约等于 3 个自然周，覆盖了一个短周期波动的完整起落。他使用的 Wilder 平滑（`1/N` 系数）比 EMA 更慢，因为他希望 RSI 过滤噪音而非追逐短期波动。

**后续演化**: 后来的交易者发现 N=14 在日线上适用，但在更短或更长的时间框架下需要调整参数。N=7（约 1.5 周）常用于短线交易，N=21–25 常用于周线。RSI 的使用也从单一的超买超卖判断扩展到背离分析（Cardwell, 1990s）和趋势确认（RSI 在牛市中持续 >40，熊市中持续 <60）。

**适用场景与局限**: RSI 在价格在一定区间内来回波动的震荡市中最有效。在强单边趋势中，RSI 会长期停留在超买区（>70）或超卖区（<30）而不回落——这被称为"钝化"，此时 RSI 的区间信号失效。限制来自 RSI 的归一化特性：如果价格连续上涨，涨幅的 Wilder 平滑持续大于跌幅，RSI 将无限逼近 100 但不会超过，导致辨别力在极端值区域下降。

### 1.2 计算公式

```
RSI_t = 100 − 100 / (1 + RS_t)

RS_t = AvgGain_t / AvgLoss_t

其中:
  Gain_t = max(Close_t − Close_{t−1}, 0)
  Loss_t = max(Close_{t−1} − Close_t, 0)

  AvgGain_t = [AvgGain_{t−1} × (N−1) + Gain_t] / N      // Wilder 平滑
  AvgLoss_t = [AvgLoss_{t−1} × (N−1) + Loss_t] / N

  初始种子: AvgGain_0 = SMA(Gain, N), AvgLoss_0 = SMA(Loss, N)
```

默认参数：**N = 14**。

### 1.3 信号含义

| RSI 读数 | 含义 |
|----------|------|
| **RSI > 70** | 过去 N 日内涨幅的 Wilder 平滑值达到跌幅平滑值的 2.33 倍以上（RS > 2.33）。这意味着近期价格上涨的持续性显著高于下跌。价格上涨的速度可能面临减速，因为维持同等速率的涨幅所需的买方力量在增大。但 RSI > 70 本身不提供何时回落的信息——在强趋势中可维持数周 |
| **RSI < 30** | 跌幅平滑值达到涨幅平滑值的 2.33 倍以上（RS < 0.43）。近期价格下跌的持续性显著高于上涨。价格下跌的速度可能面临减速。与 RSI > 70 对称，在强趋势中可长期维持 |
| **RSI 从下方突破 50** | RS 从 <1 变为 >1，即涨幅平滑值重新超过跌幅平滑值。近 N 日内的涨跌力量对比从跌方占优转为涨方占优 |
| **RSI 从上方跌破 50** | RS 从 >1 变为 <1，跌幅平滑值重新超过涨幅平滑值。涨跌力量对比从涨方占优转为跌方占优 |
| **顶背离**: 价格创新高，RSI 未创新高 | 最新一波上涨中，涨幅与跌幅的比值（RS）未能超过前一波上涨时的水平。尽管价格更高，但内部涨跌力量比在衰减——推动新高的上涨力度弱于前一次 |
| **底背离**: 价格创新低，RSI 未创新低 | 最新一波下跌中，跌幅与涨幅的比值未能超过前一波下跌时的水平。尽管价格更低，但内部跌涨力量比在衰减——打压新低的下跌力度弱于前一次 |
| **RSI 长期处于 40–60** | RS 在 0.67–1.5 之间窄幅波动，涨跌力量处于近似均衡状态。价格缺乏单向驱动力，可能处于横盘整理 |

## 2. MACD (指数平滑异同移动平均线 / Moving Average Convergence Divergence)

### 2.1 历史演化

**发明**: Gerald Appel 在 1979 年发表于《Technical Analysis of Stocks & Commodities》的系列文章中提出 MACD。Appel 是一位投资顾问，管理资金规模在当时达到数亿美元。他的核心动机是融合两个需求：既要判断趋势方向（跟踪型指标的功能），又要度量趋势力度的变化（动量型指标的功能）。

**设计思路**: Appel 选择 EMA 而非 SMA 是因为 EMA 对近期价格赋予更高权重，能更快反映趋势变化。12 和 26 这两个参数并非来自统计优化，而是当时美国股票市场的交易制度背景：12 个交易日约等于 2.5 个自然周（短周期），26 个交易日约等于 1 个月多（长周期），9 个交易日约等于 2 周（信号周期）。这些数字与 Wilder 选择 14 的逻辑类似——参数映射到自然的时间单位。

**后续演化**: 
- Thomas Aspray (1986) 发现 MACD 柱状图（BAR = DIF − DEA）比金叉死叉更早反映趋势变化——柱状图的缩短通常在交叉发生之前 1–3 根 K 线就已开始。
- 不同市场参与者在不同时间框架下调整参数：(6, 13, 5) 用于日内交易，(21, 55, 13) 用于周线级别的长期趋势跟踪。
- 背离分析（价格与 MACD 方向不一致）被加入 MACD 的使用体系，补充了金叉死叉的滞后性问题。

**适用场景与局限**: MACD 在趋势明确的市场中最有效——当价格沿一个方向持续运动时，快慢 EMA 的差值（DIF）会持续扩大，提供清晰的趋势信号。MACD 的局限在于：
1. **滞后性**: 金叉死叉基于 EMA，EMA 是对历史价格的加权平均，因此交叉信号天然滞后于价格转折点。
2. **震荡市噪音**: 当价格在窄幅区间内来回波动时，快慢 EMA 反复交叉，产生连续的假金叉和假死叉信号（拉锯效应 / whipsaw）。
3. **参数固定性**: 固定的 (12, 26) 参数假设所有品种和市场环境的交易周期一致——这在跨品种应用时可能不成立。

### 2.2 计算公式

```
EMA_t(N) = Close_t × [2/(N+1)] + EMA_{t−1} × [1 − 2/(N+1)]
  初始种子: EMA_0 = SMA(Close, N)

DIF_t = EMA(Close, fast)_t − EMA(Close, slow)_t       // 短期和长期成本的差值
DEA_t = EMA(DIF, signal)_t                             // DIF 的平滑线
BAR_t = 2 × (DIF_t − DEA_t)                             // 柱状图值

注: 部分实现使用 BAR = DIF − DEA（不乘 2），两者仅在纵轴刻度上有差异，
   不改变柱状图的正负方向和相对大小。
```

默认参数：**fast = 12, slow = 26, signal = 9**。

### 2.3 信号含义

| 信号 | 含义 |
|------|------|
| **DIF 上穿 DEA（金叉）** | DIF 的短期 EMA(DIF, 9) 被 DIF 自身突破向上，说明 DIF 正以快于其近期平均速率的速度增长。意味着短期 EMA 和长期 EMA 的间距在加速扩大，短期成本与长期成本的关系正在向有利于多头方向偏移 |
| **DIF 下穿 DEA（死叉）** | DIF 正以快于其近期平均速率的速度减小。短期 EMA 和长期 EMA 的间距在加速收缩 |
| **DIF > 0（零轴上方）** | EMA(Close, 12) > EMA(Close, 26)，近 12 日加权平均成本高于近 26 日加权平均成本。当前市场价格高于约一个月来的平均成本中枢 |
| **DIF < 0（零轴下方）** | EMA(Close, 12) < EMA(Close, 26)，近 12 日加权平均成本低于近 26 日加权平均成本。当前市场价格低于约一个月来的平均成本中枢 |
| **BAR 绝对值连续增大** | `|DIF − DEA|` 在扩大，DIF 与 DEA 的偏离程度在增加。趋势的瞬时加速度在增大，当前方向上的动能尚未衰竭 |
| **BAR 绝对值连续减小** | `|DIF − DEA|` 在缩小，DIF 正回归 DEA。趋势的瞬时加速度在减小，当前方向上的动能正在收敛。这通常比 DIF 与 DEA 的交叉提前出现 |
| **DIF 创新高而 BAR 未创新高** | DIF 的方向仍在向上，但 DIF 与 DEA 的偏离程度未同步放大。DIF 的上升速度在减慢——减速上涨 |
| **价格创新高，DIF 未创新高（顶背离）** | 最新一轮上涨中 EMA(12) 与 EMA(26) 的差值未能超过前一轮。尽管价格更高，但短期成本与长期成本的差距在缩小——上涨的内部驱动力不如前一次 |
| **价格创新低，DIF 未创新低（底背离）** | 最新一轮下跌中 EMA(12) 与 EMA(26) 的负差值未能超过前一轮。尽管价格更低，但短期成本与长期成本的负差距在缩小——下跌的内部驱动力不如前一次 |

## 3. 布林带 (Bollinger Bands)

### 3.1 历史演化

**发明**: John Bollinger 在 1980 年代早期提出布林带，最初发表于 1992 年的《Stocks & Commodities》杂志，后于 2001 年出版专著《Bollinger on Bollinger Bands》。Bollinger 曾任职于 Financial News Network（后并入 CNBC），负责技术分析内容的制作。

**解决的核心问题**: 在布林带出现之前，市场常用的通道指标是"固定百分比包络线"（如以移动平均线为中轴，上下各偏移固定百分比）。这种固定比例通道的问题是：它假设市场的波动率恒定。实际上金融市场的波动率随时间变化剧烈——在低波动时期，固定百分比通道过宽，价格永远触不到边界；在高波动时期，通道过窄，价格频繁突破边界导致大量假信号。

Bollinger 的改进是：用**标准差（σ）**替换固定百分比。标准差本身就是一个时变的波动率度量——在市场波动大时 σ 增大，通道自动变宽；波动小时 σ 减小，通道自动收窄。这使得布林带成为一个**自适应波动率的通道系统**。

**参数选择**: Bollinger 选择 N=20（约一个自然月的交易日）作为中轨周期，k=2 个标准差作为通道宽度。在正态分布假设下，2σ 区间覆盖约 95.4% 的样本。Bollinger 本人强调这个正态假设只是近似——实际金融数据的分布具有肥尾（fat tails）特征，因此价格触及轨道的频率高于 5%。他建议 k 值可以在 2.0 附近调整，但不建议低于 1.5 或高于 2.5。

**后续发展**: Bollinger 后来引入了两个辅助指标——`%b`（价格在带内的相对位置）和 `BandWidth`（带宽百分比），用于量化轨道挤压（Squeeze）和价格相对定位。Squeeze 的概念——带宽收缩至极低值后往往伴随波动率爆发——成为布林带最受关注的信号之一。

**适用场景与局限**: 布林带的核心假设是价格在一定时间尺度上围绕均值波动（均值回归倾向）。这一假设在**非趋势市场**中成立得较好。在强趋势市场中，价格可以持续沿上轨或下轨运行而不回归中轨——此时的正确解读不是"突破轨道意味着反转"，而是"价格持续沿轨道运行意味着趋势强劲"。误将趋势市中的轨道触碰当作反转信号，是布林带最常见的应用错误。

### 3.2 计算公式

```
MID_t = SMA(Close, N)_t
σ_t   = StdDev(Close, N)_t

UP_t = MID_t + k × σ_t
DN_t = MID_t − k × σ_t

辅助指标:
  BandWidth_t = (UP_t − DN_t) / MID_t × 100           // 带宽占中轨的百分比
  %b_t        = (Close_t − DN_t) / (UP_t − DN_t)       // 价格在带内的位置 [0-1区间为带内, 超出为带外]
```

默认参数：**N = 20, k = 2.0**。

### 3.3 信号含义

| 信号 | 含义 |
|------|------|
| **价格突破上轨（%b > 1）** | 收盘价超过 MID + 2σ。在当前 N 日样本中，这个价位与均值的偏差超过了 2 个标准差——属于统计意义上的极端值。说明当前价格上涨的幅度已经大到无法被近期的正常波动范围所容纳 |
| **价格跌破下轨（%b < 0）** | 收盘价低于 MID − 2σ。价格下跌到近期正常波动范围之外 |
| **带宽收窄（BandWidth 处于近期低位）** | 过去 N 日的价格波动率降至低位，价格日间变动幅度小。标准差 σ 的减小导致上下轨间距收缩。低波动率环境不会持续——历史数据显示波动率存在聚集效应（volatility clustering），但极端低波动后通常跟随波动率的均值回归（放大）。布林带将这种状态称为 Squeeze |
| **带宽从低位开始扩大** | σ 开始增大，波动率在回升。如果同时价格朝一个方向持续运行，意味着市场从低波动的均衡状态进入了有方向的高波动状态 |
| **价格连续 ≥ 5 根 K 线沿上轨运行（%b 持续 > 0.8）** | 价格持续在 MID + 1.6σ 以上的高位运行，没有回归中轨。说明这不是一次性的统计异常值，而是趋势在持续向上推进——中轨自身也在跟随上移 |
| **价格连续 ≥ 5 根 K 线沿下轨运行（%b 持续 < 0.2）** | 价格持续在 MID − 1.6σ 以下的低位运行。趋势在持续向下推进 |
| **中轨斜率由平转正** | SMA(Close, 20) 的方向从横向转为向上。近期价格的重心开始上移，N 日平均成本在升高 |
| **中轨斜率由平转负** | SMA(Close, 20) 的方向从横向转为向下。近期价格的重心开始下移 |
| **价格从上方回落至中轨附近后企稳（%b 从 >1 回落至 ≈0.5 后不再下降）** | 价格经历了突破上轨的极端运动后回归均值，但在中轨处停止下跌。中轨在这一位置起到了技术上的支撑作用——近 20 日的平均持仓成本区提供了买入兴趣 |

## 4. ATR (平均真实波幅 / Average True Range)

### 4.1 历史演化

**发明**: J. Welles Wilder Jr. 在 1978 年的《New Concepts in Technical Trading Systems》中与 RSI 一同提出 ATR。Wilder 最初设计 ATR 是为商品期货市场服务——商品期货相比股票更容易出现跳空缺口（因隔夜的天气、地缘、供需消息等因素）。

**解决的核心问题**: 传统的波动度量方式是计算当日最高价与最低价的差值（即日内振幅）。但金融市场存在大量**跳空**情况——今日开盘价与昨日收盘价之间的缺口。日内振幅无法捕捉这种跨交易日的价格跳跃。Wilder 定义了**真实波幅（True Range）**：取"日内振幅"、"今高与昨收的缺口"、"今低与昨收的缺口"三者中的最大值，确保任何形式的跨日价格运动都被纳入波动度量。

**Wilder 平滑的选择**: 与 RSI 相同，ATR 使用 `1/N` 系数的 Wilder 平滑而非 EMA 的 `2/(N+1)` 系数。Wilder 刻意选择了更慢的平滑方式，因为他的设计意图是让 ATR 反映市场波动率的背景水平（baseline），而非追踪每日波动的瞬时变化。波动率本身就具有较高的日间噪音，使用较慢的平滑可以避免 ATR 本身成为噪音源。

**后续应用扩展**: ATR 最初被 Wilder 用于他的"波动率止损"系统。后来 Van K. Tharp 在《Trade Your Way to Financial Freedom》（1998）中将 ATR 引入仓位管理——用 ATR 来标准化每一笔交易的风险暴露（仓位大小 = 可承受亏损金额 / (N × ATR)）。这一应用使 ATR 从单纯的波动度量工具扩展为风险管理的核心输入参数。

**适用场景与局限**: ATR 度量的是波动烈度而非波动方向，因此它被归类为"非方向性指标"。ATR 本身不提供任何价格走向的信息——价格上涨和价格下跌都可以伴随高 ATR。ATR 的局限在于：
1. It 仅度量历史波动率，对未来的波动预测基于"波动率短期可延续"的假设。
2. Wilder 平滑的滞后性使得 ATR 对波动率突变的响应有延迟——在波动率突然放大（如闪崩）的当天，ATR 的变化远小于当日真实波幅。

### 4.2 计算公式

```
TR_t = max(
    High_t − Low_t,                            // 当日振幅
    |High_t − Close_{t−1}|,                    // 今高与昨收的差（覆盖向上跳空）
    |Low_t − Close_{t−1}|                      // 今低与昨收的差（覆盖向下跳空）
)

ATR_t = [ATR_{t−1} × (N−1) + TR_t] / N        // Wilder 平滑
  初始种子: ATR_0 = SMA(TR, N)
```

默认参数：**N = 14**。

### 4.3 信号含义

| 信号 | 含义 |
|------|------|
| **ATR 上升** | 近 N 日的平均真实波幅在增大。无论价格上涨还是下跌，每根 K 线的价格运动幅度在扩大。市场参与者的交易活跃度和/或分歧程度在上升 |
| **ATR 下降** | 近 N 日的平均真实波幅在减小。K 线的运动幅度在缩小。市场参与者的交易活跃度降低，或市场进入多空力量暂时均衡的整理阶段 |
| **ATR 突然跳升（当前 ATR / 过去 20 日 ATR 均值 > 1.5）** | 真实波幅出现了超出近期正常范围的放大。通常与重大消息、财报发布、政策变动等事件有关——市场重新定价的过程伴随着大幅度的价格运动 |
| **ATR 持续低位（ATR / 过去 60 日 ATR 均值 < 0.5）** | 市场进入低波动环境。由于波动率与交易机会预期相关，低 ATR 阶段常被市场参与者视为"暴风雨前的平静"——这是基于波动率均值回归的经验观察：ATR 在统计上有向其长期均值回归的倾向 |
| **上涨过程中 ATR 同步放大** | 价格上升伴随波动率上升。市场上行的动力伴随着真实的资金运动——价格上涨不是低成交量的虚涨，而是有交易量支撑的活跃上涨 |
| **上涨过程中 ATR 缩小** | 价格仍在上升但波动幅度在减小。市场可能在以较低的参与度维持上涨——参与者的热情在降温，价格运动的力度在减弱 |
| **ATR / Close（归一化 ATR）** | 将 ATR 除以收盘价得到百分比形式的波动率，使不同价格水平的品种的波动烈度可以直接比较。例如 ATR=5、Close=100 的品种（ATR%=5%）比 ATR=5、Close=500 的品种（ATR%=1%）的波动烈度高 5 倍 |

## 5. KDJ (随机指标 / Stochastic Oscillator)

### 5.1 历史演化

**发明**: George C. Lane 在 1950 年代提出随机指标（Stochastic Oscillator），最初只包含 %K 和 %D 两条线。Lane 是一名期货交易员和投资教育家，曾任投资教育学会（Investment Educators）主席长达数十年，他的随机指标课程是当时最受欢迎的培训项目之一。

**解决的核心问题**: Lane 通过观察发现：在上涨趋势中，收盘价倾向于收在当日价格区间的上端（接近最高价）；在下跌趋势中，收盘价倾向于收在当日价格区间的下端（接近最低价）；在趋势即将反转时，收盘价的位置会先于价格本身开始偏移——即使价格还在创新高，收盘价相对于当日区间的位置可能已经开始下移。

随机指标将这一观察形式化为：**收盘价在过去 N 日最高价–最低价区间中的百分位位置（RSV）**。RSV 的本质是度量收盘价相对于近期波动区间的定位。

**KDJ 的由来**: 传统的 KD 指标（%K = RSV 的简单平滑，%D = %K 的简单平滑）只提供两条线。中国 A 股市场的技术分析实践者在 KD 基础上引入了 **J 线**（J = 3K − 2D = K + 2(K−D)）。J 线通过数学放大 K 与 D 的差异，比 K 和 D 更早、更剧烈地反映方向变化。J 线的名称和推广主要来自中国技术分析软件（如通达信、同花顺）——因此 KDJ 是一个在中国 A 股市场中特别流行的指标变体，国际上仍以 KD（Stochastic）为主流。

**参数演化**: Lane 的原始参数是 %K 周期为 5（即只看最近 5 根 K 线的区间），后来演化为 N=9 作为默认值（约两周交易日）。Lane 本人对参数的看法较为灵活，他认为参数应根据具体品种的波动特征调整。

**适用场景与局限**: KDJ 的高敏感度使其在**震荡市和短期波段交易**中效果最好。局限包括：
1. **高位/低位钝化**: 在强趋势中，收盘价持续收于区间上端或下端，K/D/J 长期停留在 80 以上或 20 以下且反复缠绕——此时指标的辨别力大幅下降，无法区分"趋势还在继续"和"趋势即将反转"。
2. **J 线的过度敏感**: J 线频繁超出 [0, 100] 范围，在趋势市中可以连续多日 J > 100 或 J < 0，降低了 J 线极端值作为信号的可靠性。
3. **区间依赖**: RSV 的计算完全取决于过去 N 日的最高价–最低价区间。如果区间因单根长 K 线而突然扩大，后续数日的 RSV 值都会受到该极值的影响——这被称为"锚定效应"。

### 5.2 计算公式

```
RSV_t = (Close_t − LLV_t(Low, N)) / (HHV_t(High, N) − LLV_t(Low, N)) × 100

  其中:
    LLV_t(Low, N) = 从 t−N+1 到 t 的 N 根 K 线中最低价的最小值
    HHV_t(High, N) = 从 t−N+1 到 t 的 N 根 K 线中最高价的最大值

K_t = 2/3 × K_{t−1} + 1/3 × RSV_t
D_t = 2/3 × D_{t−1} + 1/3 × K_t
J_t = 3 × K_t − 2 × D_t

  平滑理解: K = EMA(RSV, smooth_k), D = EMA(K, smooth_d)
  其中 smooth_k = 2×(1/(1/3))−1 = 5, smooth_d = 5 (当使用指数平滑等价形式时)
  初始值: K_0 = 50, D_0 = 50
```

默认参数：**N = 9, smooth_k = 3, smooth_d = 3**。

### 5.3 信号含义

| 信号 | 含义 |
|------|------|
| **RSV > 80** | 收盘价处于过去 N 日高低区间的上 80% 位置，接近区间上沿。收盘价与最高价的差距不超过区间宽度的 20% |
| **RSV < 20** | 收盘价处于过去 N 日高低区间的下 20% 位置，接近区间下沿。收盘价与最低价的差距不超过区间宽度的 20% |
| **K > 80 且 D > 80** | RSV 经过平滑后持续处于高位，说明收盘价在近期区间内的相对位置持续偏高。价格持续接近近期最高价 |
| **K < 20 且 D < 20** | RSV 经过平滑后持续处于低位，收盘价持续接近近期最低价 |
| **K 上穿 D（低位金叉）** | K 值（对 RSV 的 EMA(5)）从下方突破 D 值（对 K 的 EMA(5)）。RSV 的短期平滑线向上突破其自身的长期平滑线，说明 RSV 正在加速上升——收盘价在区间内的位置正在上移。发生在 K/D < 20 区域时，说明这一上移是从区间极低位置开始的 |
| **K 下穿 D（高位死叉）** | RSV 的短期平滑线向下突破其自身的长期平滑线，说明 RSV 正在加速下降——收盘价在区间内的位置正在下移。发生在 K/D > 80 区域时，说明这一下移是从区间极高位置开始的 |
| **J > 100** | K 的偏移量 2×(K−D) 与 K 自身之和超过了 100。K 向上偏离 D 的程度较大，且两者的方向均为向上。收盘价在区间内的位置不仅高，而且正在以加速方式上移。J > 100 的状态在数学上是不稳定的——因为 J 对 K 的放大效应在 K 增速放缓时会迅速衰减 |
| **J < 0** | K 向下偏离 D 的程度较大，收盘价在区间内的位置正在以加速方式下移。J < 0 同样是数学上不稳定的状态 |
| **价格创新高而 KDJ 未创新高（顶背离）** | 最新高点对应的收盘价在区间内的相对位置低于前一个高点——尽管绝对价格更高，但相对于最近波动区间的位置在降低。上涨的内部动量在减弱 |
| **价格创新低而 KDJ 未创新低（底背离）** | 最新低点对应的收盘价在区间内的相对位置高于前一个低点——尽管绝对价格更低，但相对于最近波动区间的位置在升高。下跌的内部动量在减弱 |

## 6. MA (移动平均线 / Moving Average)

### 6.1 历史演化

**起源**: 移动平均的最早使用可追溯到 19 世纪末的统计学家，但它在金融市场的系统应用始于 20 世纪初。最著名的早期实践者是 Richard Donchian——他在 1930 年代使用 5 日和 20 日移动平均线的交叉来生成商品期货交易信号，被公认为"趋势跟踪之父"。Donchian 的核心理念是：不要预测市场，让价格本身通过移动平均线告诉你趋势在何时发生变化。

**SMA 与 EMA 的分化**: SMA（简单移动平均）是最早的形式——每期权重相等。1960 年代，P.N. Haurlan 将指数平滑引入股票市场分析，后来 EMA 因对近期数据赋予更高权重而逐渐流行。EMA 在数学上解决了 SMA 的两个问题：(1) SMA 的"幽灵效应"——当旧数据被移出窗口时，SMA 会产生一次性的跳跃（即使当前价格不变）；(2) SMA 对所有 N 期内数据等同对待，忽略了时间衰减——昨天的价格和 N 天前的价格被赋予相同的权重，这在逻辑上不符合"近期信息更重要"的认知。

**多周期分析的发展**: 1960–70 年代，Robert Levy 和后来的 Jack Schwager 发展了多周期均线分析框架——用不同周期的 MA 同时作用，通过观察它们的排列（多头排列：短期 MA 全部在长期 MA 之上）来判断趋势的强度和多周期一致性。这一方法后来被称之为"均线系统"或"均线流"。

**适用场景与局限**: MA 的核心假设是"价格趋势具有持续性"——趋势一旦形成，会倾向于延续一段时间。这一假设在趋势市中成立，但在震荡市中失效——均线在震荡市会产生频繁的缠绕和交叉。MA 的另一个已知局限是对突发事件的响应——一次性的价格跳跃会影响后续 N 期的 SMA 值，产生"记忆效应"。

### 6.2 计算公式

```
SMA_t(N) = (P_{t−N+1} + P_{t−N+2} + ... + P_t) / N       // 等权重

EMA_t(N) = P_t × [2/(N+1)] + EMA_{t−1} × [1 − 2/(N+1)]   // 近期权重更大
  初始种子: EMA_0 = SMA(P, N)
```

### 6.3 常用周期与含义

| 周期 N | 近似时间尺度 | 信号含义 |
|--------|-------------|---------|
| **5** | 1 周 | 反映最近一周交易者的平均持仓成本。对价格变化最敏感，紧随短期价格波动。常用于判断短期动量方向 |
| **10** | 2 周 | 约半个月的平均交易成本。短期波段交易中常用的趋势参考线 |
| **20** | 1 月 | 约一个自然月的平均成本。布林带的中轨默认值。月内趋势的基准参考 |
| **60** | 1 季度 | 约一个季度的平均成本。机构投资者和中期交易者关注的水平——反映中期的市场共识价格 |
| **120** | 半年 | 半年的平均成本。长线投资者的参考线 |
| **250** | 1 年（年线） | 年度的平均成本。被广泛视为牛熊分界线——价格在年线上方运行时市场整体处于盈利状态（过去一年内入场的投资者平均在盈利），反之处于亏损状态 |

### 6.4 信号含义

| 信号 | 含义 |
|------|------|
| **多头排列**: MA5 > MA10 > MA20 > MA60（短期 MA 全部在长期 MA 之上） | 各时间尺度的平均持仓成本从短到长依次升高——所有时间框架的交易者（短期到长期）在当前价格下均处于盈利状态。不同时间尺度的市场参与者在此价格水平上形成了一致的方向共识 |
| **空头排列**: MA5 < MA10 < MA20 < MA60（短期 MA 全部在长期 MA 之下） | 各时间尺度的平均持仓成本从短到长依次降低——所有时间框架的交易者均处于亏损状态（或接近亏损）。市场整体的价格共识向下 |
| **短期 MA 上穿长期 MA**（如 MA5 上穿 MA20） | 短期平均成本从低于变为高于长期平均成本。近期参与者的持仓成本超过了较早参与者的成本——这意味着市场愿意以更高的价格入场，短期方向的共识从空转多 |
| **短期 MA 下穿长期 MA**（如 MA5 下穿 MA20） | 短期平均成本从高于变为低于长期平均成本。近期参与者以低于较早参与者的价格入场——短期方向共识从多转空 |
| **多周期 MA 间距收敛（粘合）** | 不同时间尺度的平均持仓成本趋于一致。各时间框架的市场参与者对当前价格的共识高度集中，交易活跃度往往降低。当多种时间尺度的参与者成本几乎相同时，任何方向的价格突破都会导致部分参与者开始盈利、另一部分开始亏损——这种结构的变化往往伴随趋势的启动 |
| **多周期 MA 间距发散** | 不同时间尺度的平均持仓成本差距在拉大。各时间框架的参与者之间形成了一致的趋势方向，且趋势正在加速 |
| **价格回踩 MA60 不破** | 价格从高位回调至约一季度的平均成本水平（MA60）后停止下跌。这一价格水平在过去约一季度的交易中积累了较多的成交量（持仓成本集中区），在该位置产生了较强的买入支撑 |

## 7. CCI (商品通道指数 / Commodity Channel Index)

### 7.1 历史演化

**发明**: Donald Lambert 在 1980 年发表于《Stocks & Commodities》杂志的《Commodity Channel Index: Tool for Trading Cyclic Trends》一文中提出 CCI。Lambert 的职业是期货交易员和技术分析师。CCI 最初是为商品期货市场设计的——商品价格具有较强的周期性特征（季节性供需周期），Lambert 希望量化价格在其周期性波动中的"异常偏离程度"。

**设计思路**: Lambert 没有选择标准差来度量偏离（如布林带那样），而是选择了**平均绝对偏差（Mean Deviation）**。他的理由是：标准差对方差放大效应敏感——一根极端 K 线会不成比例地拉大标准差，而平均绝对偏差对极值的敏感度较低，能更稳健地反映价格的"正常偏离程度"。

**0.015 常数的意义**: CCI 公式中的常数 0.015 是 Lambert 通过回测选择的，目的是让约 70–80% 的 CCI 读数落在 [−100, +100] 区间内。这个常数没有统计学上的必然性——它是经验校准的结果，使 CCI 在商品期货的日线数据上有良好的辨别力。

**从商品到股票的迁移**: CCI 虽名为"商品通道指数"，但后来被广泛用于股票市场。股票的趋势不如商品有季节性周期性，但 CCI 度量统计异常偏离的特性在股票市场上同样有效——它量化的是"当前价格偏离其近期典型价格（TP）的统计显著程度"。

**适用场景与局限**: CCI 最大的特征是**无界**——与 RSI [0, 100] 不同，CCI 理论上可以趋向任意大的正值或负值。这使得 CCI 在强趋势中不会像 RSI 那样钝化——当 RSI 已经在 85 附近无力辨别时，CCI 可能已经到了 +300，继续反映趋势的"极端程度"。局限包括：
1. 平均绝对偏差对波动率变化的响应是非线性的——它不完全等价于标准差替代方案。
2. 0.015 常数是在 1980 年代的期货市场上优化的，在今天的股票市场上是否仍然最优缺乏验证。

### 7.2 计算公式

```
TP_t = (High_t + Low_t + Close_t) / 3                  // 典型价格 (Typical Price)

MA(TP)_t = SMA(TP, N)_t                                // N 期典型价格的简单移动平均

MD_t = SMA(|TP − MA(TP)|, N)_t                          // 平均绝对偏差 (Mean Deviation)

CCI_t = (TP_t − MA(TP)_t) / (0.015 × MD_t)

  其中 MD 是 TP 与 MA(TP) 的差值的绝对值在过去 N 期的平均值
```

默认参数：**N = 20**。

### 7.3 信号含义

| 信号 | 含义 |
|------|------|
| **CCI > +100** | 典型价格（TP）高于其 20 期均值超过 1.5 倍的平均绝对偏差（0.015 × 100 = 1.5）。这意味着当前 TP 偏离其近期均值的幅度已超过正常波动范围的 1.5 倍——处于统计意义上的强势区域 |
| **CCI < −100** | 典型价格低于其 20 期均值超过 1.5 倍的平均绝对偏差。当前 TP 处于统计意义上的弱势区域 |
| **CCI > +200** | TP 偏离均值超过 3 倍的平均绝对偏差。属于极端偏离——在当前波动率水平下，这种幅度的正向偏离在近期内属于罕见事件 |
| **CCI < −200** | TP 偏离均值超过 3 倍平均绝对偏差的负向偏离。同等程度的极端负偏离 |
| **CCI 从下方突破 +100** | TP 从正常波动范围进入统计强势区域。这是 TP 偏离均值的幅度首次超过 1.5 倍 MD |
| **CCI 从上方跌破 +100** | TP 从强势区域回归正常波动范围。TP 偏离均值的幅度缩减至 1.5 倍 MD 以内——统计显著性降低 |
| **CCI 从下方突破 −100** | TP 从弱势区域回归正常波动范围 |
| **CCI 在 +100 以上持续** | TP 持续高于其 20 期均值的 1.5 倍 MD。说明价格在持续地、有统计显著性地高于近期均值——趋势具有持续性 |
| **CCI 在 [−100, +100] 内震荡** | TP 在其 20 期均值的 ±1.5 倍 MD 范围内运行。当前价格运动在统计上属于正常波动范围，无明显的异常偏离 |

## 8. ADX (平均趋向指数 / Average Directional Index)

### 8.1 历史演化

**发明**: J. Welles Wilder Jr. 在 1978 年的《New Concepts in Technical Trading Systems》中与 RSI 和 ATR 一同提出。ADX 是 Wilder 的"趋向运动系统"（Directional Movement System）的核心组成部分。Wilder 可能是第一个将"趋势是否存在"作为一个可量化的问题来研究的人。

**设计思路的分离思想**: Wilder 在设计 DMS 时做了一个关键的概念分离：**方向**（价格上涨还是下跌）和**趋势强度**（无论方向如何，价格是否在朝一个方向持续运动）是两个独立的问题。他设计了三个互相配合的指标：
- **+DI**（正向趋向指标）：度量向上的方向运动
- **−DI**（负向趋向指标）：度量向下的方向运动
- **ADX**（平均趋向指数）：度量方向运动的强度，不区分向上还是向下

这种分离是技术分析中的一个重要创新——在此之前，趋势指标普遍将"方向"和"强度"混在一起度量。ADX 使得交易者可以回答一个此前未被系统回答的问题："**现在是否适合使用趋势跟踪策略？**"——当 ADX < 20 时，无论 ±DI 如何交叉，趋势跟踪策略的可靠性都会降低。

**DX 到 ADX 的平滑**: Wilder 先计算 DX（Directional Index = |(+DI) − (−DI)| / ((+DI) + (−DI)) × 100），再用 Wilder 平滑将 DX 转化为 ADX。这个平滑步骤至关重要——单日的 DX 波动极大（因为它依赖 ±DI 的比值，±DI 本身又是真实波幅的函数），不经平滑的 DX 几乎没有实用价值。

**后续发展**: ADX 后来被广泛应用于非商品市场（股票、外汇、债券），并衍生出 ADXR（ADX 的平滑形式）等变体。Alexander Elder 和 Charles LeBeau 等人在 1990 年代进一步发展了 ADX 的使用框架，将 ADX 值与具体的市场状态对应起来（<20 震荡，20–25 过渡，25–40 趋势，>40 强趋势）。

**适用场景与局限**: ADX 解决的核心问题是"趋势是否存在"，因此它最适用于帮助决定应该使用震荡策略还是趋势跟随策略。局限包括：
1. ADX 是一个**延迟确认**指标——当前的 ADX 值反映的是过去 N 期平均的趋势强度。当 ADX 确认趋势存在时，趋势可能已经运行了大半。
2. ADX 无法预测趋势何时结束——ADX 从高位回落只能说明"趋势强度在减弱"，但无法区分"趋势暂时休整"和"趋势彻底反转"。
3. 在突发事件（如重大政策公告）导致的价格急剧运动中，ADX 的响应会落后于当日价格运动。

### 8.2 计算公式

```
// 第一步：计算趋向运动 (Directional Movement)
+DM_t = max(High_t − High_{t−1}, 0)   // 正向趋向运动
−DM_t = max(Low_{t−1} − Low_t, 0)     // 负向趋向运动

  条件过滤: 如果 +DM_t > −DM_t 且 +DM_t > 0, 则 −DM_t = 0
           如果 −DM_t > +DM_t 且 −DM_t > 0, 则 +DM_t = 0
           如果 +DM_t = −DM_t 或两者均 ≤ 0, 则两者均 = 0
  // 这个过滤的逻辑是: 在一根K线内, 只保留占主导的趋向方向

// 第二步：计算平滑趋向运动和 ATR
+DI_t = Wilder(+DM, N)_t / ATR_t × 100     // ATR 由同一套 Wilder 平滑公式计算
−DI_t = Wilder(−DM, N)_t / ATR_t × 100

// 第三步：计算方向运动指数
DX_t = |(+DI_t) − (−DI_t)| / ((+DI_t) + (−DI_t)) × 100

// 第四步：平滑得到 ADX
ADX_t = Wilder(DX, N)_t                     // Wilder 平滑

  初始种子: Wilder(DM, N)_0 = SMA(DM, N), ADX_0 = SMA(DX, N)
```

默认参数：**N = 14**。

### 8.3 信号含义

| 信号 | 含义 |
|------|------|
| **ADX > 25** | 过去 14 日的趋向运动强度经过平滑后处于中等偏上水平。市场存在可识别的方向性运动——价格在朝某个方向持续运动，而非在区间内来回摆动。通常情况下 ADX > 25 被视为"趋势存在"的阈值 |
| **ADX > 40** | 趋向运动强度处于高位。市场正在经历强趋势——无论涨跌，价格朝一个方向的运动非常显著且持续 |
| **ADX < 20** | 趋向运动强度不足。过去 14 日中 ±DI 的差值相对于它们的总和较小——+DI 和 −DI 数值接近，多空方向力量交替占优。市场缺乏持续的方向性驱动力，可能正在横盘整理 |
| **ADX 从 <20 向上突破 25** | 趋向运动强度从低位进入中等水平。市场的方向性从"不可识别"变为"可识别"——某种方向的力量（涨或跌）正在从震荡中浮现，并开始建立持续性 |
| **ADX 从高位（>40）开始回落** | 趋势强度正在减弱。这不直接等于趋势反转——趋势强度的衰减可能是趋势暂停（横盘整理后继续原方向）或趋势逆转的前奏。ADX 回落本身不提供方向信息 |
| **+DI > −DI 且 ADX > 25** | 正向趋向运动占优 + 趋势存在。在可识别的趋势中，向上力量高于向下力量。+DI 和 −DI 的差值越大，向上的主导性越强 |
| **−DI > +DI 且 ADX > 25** | 负向趋向运动占优 + 趋势存在。在可识别的趋势中，向下力量高于向上力量 |
| **+DI 上穿 −DI** | 正向趋向运动从低于负向变为高于负向。方向的主导权从空方转移到多方。这个交叉本身不提供趋势强度的信息——如果 ADX < 20 时发生交叉，方向的转变可能短暂且不可靠 |
| **−DI 上穿 +DI** | 方向的主导权从多方转移到空方 |
| **ADX 上升同时 +DI 和 −DI 都上升** | 一个不常见但值得注意的配置：多空双方的力量都在增强。这意味着市场波动率在增大、多空争夺激烈，但尚未决出明确的主导方向 |

## 9. 指标互补关系

八个指标从不同的数据源和数学变换出发，度量市场的不同维度。将它们按观察维度分组有助于理解指标之间的信息重叠和互补关系：

### 9.1 维度分类

| 观察维度 | 回答的问题 | 核心指标 | 辅助指标 |
|----------|-----------|---------|---------|
| **趋势方向** | 价格重心在向哪个方向移动？ | MA 排列、MACD (DIF 正负) | 布林带 (中轨斜率) |
| **趋势强度** | 方向性运动有多强？ | ADX | MACD (BAR 柱大小) |
| **动量/速度** | 涨跌的力有多大？ | RSI (涨跌力量比) | KDJ (收盘价位置变化速度) |
| **波动烈度** | 价格波动有多大？ | ATR | 布林带 (带宽) |
| **极端位置** | 价格是否处于统计边缘？ | 布林带 (%b, 轨道) | CCI (±100/200) |
| **短期拐点** | 动量即将减速还是加速？ | KDJ (K/D/J 交叉) | MACD (BAR 缩短), CCI (±100 穿越) |

### 9.2 不同市场状态下的有效指标

**震荡市**（ADX < 20，布林带宽窄）：
价格在一定区间内来回运动，无单边趋势。此时最有效的信息维度是**极端位置**和**短期动量拐点**——指标能识别价格何时接近区间边界以及动量何时开始向反方向偏移。RSI 的 30/70 区域信号、KDJ 的 20/80 交叉信号、CCI 的 ±100 穿越信号、布林带的轨道触碰信号在此类市场中提供的信息量最大。MACD 的金叉死叉在震荡市中假信号率高，因为快慢 EMA 反复小幅交叉。

**趋势市**（ADX > 25，布林带宽扩）：
价格朝一个方向持续运动。此时**趋势方向和强度**的度量最为关键，而超买超卖类指标（RSI、KDJ）会发生钝化，极端位置信号不可靠。MA 排列（多头/空头）和斜率、MACD 的 DIF 正负和 BAR 方向、ADX 的强度读数、布林带中轨斜率在此类市场中提供的信息量最大。ATR 的升降可用于判断趋势是否伴随真实波动——上涨+ATR 放大意味着趋势健康。
