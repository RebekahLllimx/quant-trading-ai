# 数据炼金术 — 8 大技术指标计算与交互看板

> **核心交付**：8 个技术指标的完整 Python 计算引擎 + 自包含交互看板，覆盖趋势/动量/波动/极端位置四个维度。每个指标可从看板中调节参数、切换标的、实时查看信号变化。

---

## 📖 Overview（项目概述）

- **项目类型**：技术指标计算引擎 + 可视化看板
- **解决的问题**：将 8 个经典技术指标的计算公式精确实现为 Python 代码，并提供交互式看板让用户直观理解每个指标如何随参数和标的变化
- **核心能力**：Wilder 平滑 vs EMA 的区分、冷启动截断处理、多指标并行计算

## 📐 Indicators（8 个指标）

### 趋势方向

| 指标 | 公式 | 默认参数 |
|------|------|---------|
| **MA（移动平均线）** | `SMA(N) = mean(Close_{t-N+1}, ..., Close_t)` | 5, 10, 20, 60, 120, 250 |
| **MACD** | `DIF = EMA(C,12) − EMA(C,26)`, `DEA = EMA(DIF,9)`, `BAR = 2×(DIF−DEA)` | 12/26/9 |

### 动量/速度

| 指标 | 公式 | 默认参数 |
|------|------|---------|
| **RSI（相对强弱指数）** | `RSI = 100 − 100/(1+RS)`, `RS = AvgGain/AvgLoss`（Wilder 平滑 α=1/14） | 14 |
| **KDJ（随机指标）** | `RSV = (C−L₉)/(H₉−L₉)×100`, `K/D = EMA(RSV)`, `J = 3K−2D` | 9/3/3 |

### 波动烈度

| 指标 | 公式 | 默认参数 |
|------|------|---------|
| **ATR（平均真实波幅）** | `TR = max(H−L, |H−Cₚ|, |L−Cₚ|)`, Wilder(TR, 14) | 14 |
| **布林带（Bollinger Bands）** | `MID = SMA(C,20)`, `UP/DN = MID ± 2×σ` | 20 / 2.0 |

### 极端位置 / 趋势强度

| 指标 | 公式 | 默认参数 |
|------|------|---------|
| **CCI（商品通道指数）** | `CCI = (TP − SMA(TP,N)) / (0.015 × MD)` | 20 |
| **ADX（平均趋向指数）** | `DX = |+DI−(−DI)| / (+DI+(−DI))×100`, `ADX = Wilder(DX,14)` | 14 |

### 关键实现细节

- **Wilder 平滑 α=1/N**（RSI/ATR/ADX），非 EMA 的 α=2/(N+1)——三个指标必须区分
- **冷启动截断**：在全量数据上计算指标，仅显示时按回看天数截取尾部——避免前期 NaN 导致指标曲线被压缩
- **null→0 污染**：EMA/Wilder 平滑从第一个有效索引开始计算，前面补 null，避免前期 null 被当成 0 参与平滑

## 💡 Key Findings（关键发现）

1. **指标互补关系**：8 个指标分属四个维度（趋势方向、动量速度、波动烈度、极端位置），不同市场状态下有效指标不同。震荡市（ADX<20）：RSI 30/70、KDJ 20/80、布林带轨道触碰有效；趋势市（ADX>25）：MA 排列、MACD 方向、ADX 强度有效。
2. **Wilder 平滑不可用 EMA 替代**：Wilder 的 α=1/14 ≈ 0.071，EMA(14) 的 α=2/15 ≈ 0.133——后者对近期数据的权重几乎是前者的两倍，导致 RSI/ATR/ADX 对噪音更敏感。
3. **参数选择依赖于时间框架和市场特征**：没有"最优参数"——Wilder 选择 14 是基于 1970 年代商品期货的交易周期（3 个自然周），在 A 股市场和不同时间框架下需要重新评估。
