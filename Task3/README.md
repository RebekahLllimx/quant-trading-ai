# 双均线交叉策略 — 金叉买入、死叉卖出，全仓进出

> **核心结论**：双均线策略是最基础的趋势跟踪策略。简单是它的优势（逻辑透明、参数少、不易过拟合），也是它的局限（信号单一、震荡市反复亏损）。作为一个入门策略，它的价值不在于盈利能力，而在于建立了完整的"策略定义→回测→评估→参数分析"方法论框架。

---

## 📖 Strategy Overview（策略概述）

- **策略类型**：趋势跟踪（Trend Following）
- **核心思想**：短周期均线（MA5）代表近期价格重心，长周期均线（MA15）代表中期趋势背景。当短期均价上穿长期均价（金叉）→买入；当短期均价下穿长期均价（死叉）→卖出。不做空，不设止损，纯多头。
- **历史背景**：均线交叉是最古老的机械交易系统之一。Richard Donchian 在 1930 年代首次系统使用 5/20 日均线交叉生成商品期货信号，被公认为"趋势跟踪之父"。
- **适用市场**：单边趋势市场。不适用于区间震荡市场（反复交叉产生假信号）。

---

## 📐 Mathematical Definition（数学定义）

### 简单移动平均（SMA）
```
SMA_N(t) = (Close_{t-N+1} + ... + Close_t) / N
```

### 交易信号（避免未来函数）
```
金叉（买入）：MA_short[t-1] > MA_long[t-1] AND MA_short[t-2] ≤ MA_long[t-2]
            → 以 Close[t] 全仓买入

死叉（卖出）：MA_short[t-1] < MA_long[t-1] AND MA_short[t-2] ≥ MA_long[t-2]
            → 以 Close[t] 全部卖出
```
- 信号判断使用 T-1 和 T-2 日均线值（T 日开盘前已知）
- T 日收盘价执行交易
- 首次信号出现前持有现金

### 交易成本
- 手续费：0.03%（万三），买卖各收一次
- 滑点：0.01%（买入上浮、卖出下浮）
- 单边总成本 ≈ 0.04%

## ⚙️ Parameters（参数说明）

| 参数 | 默认值 | 范围 | 作用 |
|------|--------|------|------|
| 短均线周期 | 5 | 2–60 | 控制信号的敏感度——越短信号越多 |
| 长均线周期 | 15 | 5–120 | 控制趋势确认的周期——越长信号越少 |
| 初始资金 | 1,000,000 | 10,000–100,000,000 | 回测起始资金 |
| 手续费率 | 0.03% | 0–1% | 买卖各收一次 |
| 滑点率 | 0.01% | 0–1% | 买入上浮、卖出下浮 |

## 📊 Backtest Results（回测结果）

**回测设置**：贵州茅台 (600519) / 2025-05-29 ~ 2026-07-03 / 初始资金 100 万 / MA5+MA15

| 指标 | 双均线策略 | 买入持有 |
|------|:------:|:------:|
| 累计回报 | −5.67% | −18.24% |
| 年化收益 | −3.61% | −11.63% |
| 最大回撤 | −15.20% | −21.35% |
| 夏普比率 | −0.58 | −1.12 |
| 胜率 | 35.7% | — |
| 交易次数 | 14 | — |

**解读**：在 13 个月的下跌市中，策略跑赢买入持有约 12.6 个百分点，但绝对收益为负——仅做多的趋势跟踪策略在单边下跌市场中无法盈利。14 次交易中约 35% 盈利，盈亏比约 2.4——少数几次盈利覆盖了多数小亏。

## 📈 Parameter Sensitivity（参数敏感性）

对短均线（3/5/10/15/20/30）× 长均线（10/15/20/30/40/60）36 组参数扫描：

- **短周期与长周期的间隔太小**（如 MA3+MA10）：反复交叉，交易次数激增，手续费累积吞噬利润
- **间隔适中**（如 MA5+MA15、MA10+MA30）：平衡信号可靠性与响应速度
- **间隔过大**（如 MA5+MA60）：信号过少，错过大部分趋势
- **夏普比率热力图比回报热力图更平滑**：参数选择应优先参考风险调整后收益

## 💡 Key Findings（关键发现）

**什么时候有效**：
- 价格呈现持续单向运动，均线方向明确向上或向下
- 趋势持续时间 ≥ 长均线周期的 2–3 倍

**什么时候失效**：
- 区间震荡——价格在均线附近反复穿越，产生连续假信号
- 趋势初期——金叉确认时趋势已运行一段，入场偏晚
- 急跌行情——死叉确认时已在底部附近，卖出偏晚

**策略评价**：
- 双均线是"最小可行趋势跟踪策略"——只用两个参数、一个指标（收盘价）、一条规则
- 其简单性本身就是一种优势：参数空间小 → 过拟合风险低 → 样本外更可能保持稳定
- 不适合单独用于实盘——缺少止损机制、仓位管理、趋势过滤器——但作为入门策略和策略比较基线非常有价值

---

## 🚀 Future Improvements（改进方向）

1. **ADX 趋势过滤器**：ADX < 20 时暂停交易，减少震荡市中的假信号
2. **加入止损**：单笔止损（如 −5%）或移动止损（如 −2×ATR），控制单次亏损上限
3. **仓位管理**：从全仓进出改为基于波动率的仓位公式，使每笔交易风险恒定
4. **多周期确认**：增加 MA60 作为长期趋势过滤器，仅在长期方向与交易方向一致时执行
